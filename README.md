## About

This repository contains a series of projects completed during MERN stack internship training. Each project demonstrates different aspects of web development, from frontend UI components to backend APIs and data management. Projects include both vanilla JavaScript implementations and TypeScript conversions for better type safety and maintainability.

**Short Description:**  
*A collection of full-stack JavaScript/TypeScript projects including API dashboards, form validators, shopping cart systems, and CLI applications—perfect for learning modern web development patterns and best practices.*

---

## Projects Overview

### 1. **API Data Dashboard** 📊
A dynamic dashboard that fetches and displays data from APIs with advanced features.
- **Features:** Infinite scroll, search functionality, comment system, user profiles
- **Tech Stack:** HTML, CSS, JavaScript, Fetch API, Caching
- **Location:** `MERN I/API data dashboard/`

### 2. **Bank Account Simulator** 🏦
A Node.js application simulating bank account operations using the Factory pattern.
- **Features:** Account creation, transactions, balance management
- **Tech Stack:** Node.js, JavaScript (ES6+)
- **Location:** `MERN I/Bank Account Simulator/`

### 3. **Dynamic Form Validator** ✅
A client-side form validation system with real-time feedback and storage.
- **Features:** Field validation, debounce/throttle, local storage, dynamic UI updates
- **Tech Stack:** HTML, CSS, JavaScript, Local Storage API
- **Location:** `MERN I/Dynamic form validator/`

### 4. **Modular Shopping Cart** 🛒
A modular shopping cart system with product management and database simulation.
- **Features:** Add/remove items, inventory management, cart calculations
- **Tech Stack:** Node.js, JavaScript, JSON database
- **Location:** `MERN I/Modular shopping cart/`

### 5. **Shopping Cart TS Conversion** 🛒📘
TypeScript version of the modular shopping cart with enhanced type safety.
- **Features:** Same as modular version with full TypeScript support
- **Tech Stack:** Node.js, TypeScript, JSON database
- **Location:** `MERN I/Shopping cart TS conversion/`

### 6. **Task Tracker CLI APP** ✏️
A command-line application for managing tasks with file-based persistence.
- **Features:** Add/delete/list tasks, date utilities, file operations, JSON storage
- **Tech Stack:** Node.js, JavaScript, File System API
- **Location:** `MERN I/Task Tracker CLI APP/`

### 7. **Task Tracker CLI TS Conversion** ✏️📘
TypeScript version of the CLI task tracker with improved code structure.
- **Features:** Task management, file handling, menu-driven interface
- **Tech Stack:** Node.js, TypeScript, File System API
- **Location:** `MERN I/Task tracker CLI to TS conversion/`

## Getting Started

### Prerequisites
- Node.js (v14 or higher)
- npm or yarn
- Basic knowledge of JavaScript/TypeScript

### Installation

1. Clone the repository:
```bash
git clone https://github.com/aliza539/mern-intern-tasks.git
cd mern-intern-tasks
```

2. Navigate to any project directory:
```bash
cd "MERN I/Bank Account Simulator"
```

3. Install dependencies:
```bash
npm install
```

4. Run the project:
```bash
node src/demonstration.js       # For Node.js projects
npm start                       # If scripts are configured
```
